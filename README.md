# Myebb

A SwiftUI-based iOS application for mood tracking and mental wellness.

## Features

- **Mood Logging**: Track your daily mood and emotional state
- **Mood History**: View your mood patterns over time
- **User Authentication**: Secure login using Google Sign-In
- **Dashboard**: Overview of your mood tracking progress
- **User Profiles**: Personalized user experience

## Tech Stack

- **Framework**: SwiftUI
- **Platform**: iOS
- **Authentication**: Google Sign-In
- **Language**: Swift

## Security

- OAuth credentials are stored in `.plist` files that are excluded from version control
- User authentication is handled securely through Google Sign-In
- No sensitive data is hardcoded in the source code
